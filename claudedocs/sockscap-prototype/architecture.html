目标架构与平台数据路径

一个系统捕获面、多个有序配置组、统一 PolicyEngine，以及可插拔的 Direct / SOCKS5 / HTTP CONNECT / SSH Jump 出站连接器。

Architecture · Revision 2
✓
架构原则 平台 helper 只捕获流量和恢复网络；产品规则、凭据解析、SSH channel 与统计都留在非特权 Rust core。helper 不接触 Proxy/SSH secret。

产品边界
selected + route 的后置过滤能力不足时必须可解释降级，不声称只捕获选中程序。
MITM inspectionSockscap 不安装根证书、不解密 HTTPS、不保存 body。
仅依赖正常退出 cleanup补 helper heartbeat、recovery journal、下次启动修复。
未经审核的 WinDivert 打包LGPLv3/GPLv2、签名驱动和本地 patch 必须先过 Gate。
