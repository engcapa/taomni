内网通讯 LanChat交互原型

面向局域网团队的去中心化即时通讯模块,灵感取自"飞鸽传书",但更现代统一。无中心服务器,基于 mDNS 自动发现 + 节点间 P2P 直连,内置文字、文件、音视频会议与协作白板。下方原型与 Taomni 主窗口 视觉体系保持一致(靛蓝主色 / Inter 字体 / 明暗双主题),并演示弹出独立窗口与 QQ 式四面边缘抽屉两种窗口形态。

去中心化 P2PmDNS 自动发现 弹窗独立四面抽屉式停靠 明暗双主题无服务器
点击右上角 ◐ 切换明暗。卡片点击进入对应原型;主原型内可体验"弹出独立窗口"与"边缘抽屉停靠"。

核心通讯(主原型)

嵌入 Taomni 窗口的成员名册、会话、群组、在线状态、@提及、文件卡片。含"弹出独立窗口"与"四面边缘抽屉"交互。
03打开 ↗

音视频会议

WebRTC mesh 多人会议:画面平铺、说话人高亮、屏幕共享大格、麦克风/摄像头/挂断控制。可拖动的独立窗口。
04打开 ↗

协作白板

可真实绘制的多人白板:画笔/形状/箭头/便签/橡皮擦,多人光标、Yjs CRDT 同步指示。可在会议中调出。

原型覆盖的能力

1:1 与群组消息私聊 + 命名频道,网状扩散,本地 SQLite 历史持久化。
在线状态 / 身份显示名 + 头像 + 状态签名,在线/离开/忙碌/离线,mDNS 实时更新。
@提及 + 通知@ 高亮、未读角标、桌面通知。
文件 / 截图传输P2P 直传,进度/暂停/取消,图片内联预览。
音视频会议WebRTC mesh,屏幕共享,说话人指示,无 STUN/TURN。
协作白板Yjs CRDT 实时协作,多人光标,会议内调出。
弹出独立窗口会话脱离主窗成为可拖动浮窗(桌面版为真实 WebviewWindow)。
四面边缘抽屉停靠到屏幕上/下/左/右,鼠标移出自动收起,边缘标签点击弹出(QQ 式)。
