操作指南（评审清单）

对应设计文档 §3.1，逐项体验。面板尺寸固定为演示值。
  • 布局：工具条右侧 ⬅⬇➡ 折叠三区
  • 文件树：单击=预览打开（斜体 tab），双击=固定；右键菜单
  • 编辑器 tab：右键菜单（固定/关闭其他…），✕ 关闭
  • Search Everywhere：双击 Shift 或点顶部搜索框；Tab 切分组；输入 cwt 试 camelCase 匹配
  • 编辑器内查找：Ctrl+F，输入即高亮计数
  • 语言智能（虚线下划线的标识符）：
    • 悬停 → 快速文档（右上可 pin 到右栏）
    • 单击 → 同符号用法高亮
    • 右键 → 跳转声明/查找用法/调用层级/类型层级/重命名/快速文档
  • Quick Fix：波浪线所在行 → 行首 💡 或点击行后 Alt+Enter
  • 点击代码行：活动行高亮 + 状态栏行列 + 面包屑符号联动
  • 底部面板：Problems/Find in Files（可真实搜索假数据）/Terminal/Run/References/调用层级（方向切换+懒展开）/类型层级
  • Git：gutter 绿/蓝色条与红三角；工具条 👤 开关 inline blame
  • 分屏：工具条 ◫；导航历史：←/→
  • 快捷键：Alt+F12 底部开关，Esc 关闭浮层
